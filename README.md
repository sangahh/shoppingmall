# 쇼핑몰 프로젝트
### SpringBoot와 JPA를 이용한 쇼핑몰입니다.
### 로그인/로그아웃, 상품등록, 상품관리, 장바구니, 주문하기/주문취소, 구매이력 등 쇼핑몰에 필요한 핵심 기능 구현

## 사용기술
+ SpringBoot 2.5.9, Thymeleaf, JPA, MySQL5.8.9, 
+ Java11
+ Maven
+ IntelliJ
